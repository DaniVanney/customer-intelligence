# Customer Intelligence

Plataforma de inteligencia de clientes construida a partir de datos
transaccionales públicos.

## Objetivo

Ayudar a un responsable comercial a priorizar clientes, detectar riesgo de
inactividad y anticipar ventas futuras.

## Módulos

1. Customer Scoring mediante RFM.
2. Predicción de riesgo de inactividad.
3. Forecasting semanal de ventas.
4. Aplicación interactiva en Streamlit.
